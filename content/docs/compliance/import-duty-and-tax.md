# Import Duty and Tax

## HTS classification
- The HTS code determines the duty rate. Misclassification is the most
  common compliance error and the most expensive.
- For ambiguous classifications, request a binding ruling from customs
  before quoting.

## Valuation
- Dutiable value is the transaction value at the time of export, plus
  any assists, royalties, and packing costs.
- Do not include post-importation costs (duty, freight, brokerage) in
  the dutiable value.

## Duty minimization
- Free trade agreements (USMCA, EU FTAs, etc.) may zero out the duty if
  the goods qualify. Always check before quoting.
- First sale for export is allowed for US imports when the transaction
  structure supports it.

## Recordkeeping
All import records are kept for 5 years from the date of entry.
