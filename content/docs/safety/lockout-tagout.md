# Lockout/Tagout

## When
Any maintenance, repair, or cleaning of a machine where the
unexpected start-up or release of stored energy could cause injury.

## Procedure
1. Notify all affected employees.
2. Shut down the machine using the normal stop procedure.
3. Isolate the machine from its energy source (electrical, hydraulic,
   pneumatic, mechanical, thermal).
4. Apply a personal lock and tag to each isolation point.
5. Release any stored energy.
6. Verify the isolation by attempting to start the machine; the
   machine must not start.

## Removal
Only the employee who applied the lock may remove it. If that
employee is unavailable, the manager follows a documented procedure
to remove the lock, including a thorough inspection of the work
area.

## Training
All authorized employees receive initial training and annual
refresher training. The training is documented in the LMS.
